# Number Guessing AI

A simple Python game where an AI tries to guess the number you are thinking of using a binary search algorithm.

## Features
- AI guessing using binary search
- Multiple difficulty levels (1–50, 1–100, 1–500)
- Tracks number of attempts
- Performance rating system
- Simple AI-style messages

## Future Improvements
- Add GUI version
- Add difficulty auto-adjustment
- Store stats in file (persistent storage)

## How to Run
python main.py
